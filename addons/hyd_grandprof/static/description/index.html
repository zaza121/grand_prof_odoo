Control and set salary of your staff easyly by using salary grid

By using this module you will be able to:

  • 1) Save all your salary grid
  • 2) Update all contract wage just by updating the wage of the grid
  • 2) See all your monthly payroll and identify which salary grid has the bigger part of the payroll
